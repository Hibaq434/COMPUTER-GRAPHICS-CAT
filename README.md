# Computer Graphics CAT — Residential Building Design





## Project Overview

This project is a computer graphics assignment involving the architectural design and visualization of a multi-story residential apartment building. It demonstrates proficiency in 2D drafting, building systems planning, and 3D architectural visualization using professional CAD/design software.



## Project Files

| File | Description |
|------|-------------|
| `COMP GRAPHICS  group work .pln` | Main editable project file (GSPublisher/ArchiCAD format) containing all design layers and data |
| `COMP GRAPHICS new file.pdf` | Utility and infrastructure systems plans (Wi-Fi, Electrical, Water) |
| `GRAPHICS WORK.pdf` | Full architectural drawing package — floor plans, elevations, and 3D views |

---

## Building Design Details

- **Building Type:** Multi-story residential apartment building
- **Scale:** 1:100
- **Overall Dimensions:** Approximately 19,500mm x 16,900mm per floor
- **Floors:** Ground floor + upper stories (up to 19th floor referenced)
- **Units Per Floor:** Multiple residential units
- **Unit Layout:** Master bedroom, bedroom(s), lounge, kitchen, bathroom (WC), shower, balcony
- **Shared Features:** Centrally located elevator system


## Drawings Included

### Architectural Plans (`GRAPHICS WORK.pdf`)
- Ground Floor Plan (1:100)
- Typical Story Plan — 13th Floor (1:100)
- Building Elevations: E-01, E-02, E-03, E-04 (1:100)
- Detailed unit floor plans with furniture layouts and dimension annotations
- 3D perspective views (front, side, and isometric)

### Building Systems Plans (`COMP GRAPHICS new file.pdf`)
- **Wi-Fi / Network Plan:** Router placement and wireless coverage zones
- **Electrical Plan:** Power lines, switch lines, and distribution board locations
- **Water/Plumbing Plan:** Supply lines, drainage/sewage lines, and pump locations

## Software Used

- **GSPublisher / ArchiCAD** — Primary design and drafting software
- **PDF export** — For final drawing submission



## Group Work Note

The `.pln` file is titled "group work", indicating this was a collaborative project. All architectural and systems drawings were produced as a team.



## How to Open the Project File

1. Install **ArchiCAD** or **GSPublisher** (compatible version).
2. Open `COMP GRAPHICS  group work .pln` from within the application.
3. Navigate the layers panel to toggle between architectural, electrical, plumbing, and network plans.

To view the submitted drawings without the software, open either PDF file using any standard PDF viewer.
